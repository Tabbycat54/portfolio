CSS Positioning Exercise

This is a positioning exercise to differentiate between the four different types of CSS Positioning:

  • Static: HTML Default Flow
  • Relative: Position Relative to Default Position
  • Absolute: Position relative to the nearest position ancestor or the top left corner of the webpage
  • Fixed: Position relative to the top left corner of the browser and moves with the webpage as it scrolls.
