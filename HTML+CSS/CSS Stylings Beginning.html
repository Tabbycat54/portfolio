Three Methods of Adding CSS

  • Inline - This is where you are inline with the html code. This is not ideal for more than one element.
  • Internal - This is where you put the style in the head. Useful for one page styling, not for multiple pages.
  • External - This is where you put the CSS in a separate file and link it to your HTML document.
